# Python - Lists and Sequences

## Project Overview

This project focuses on Python **lists, tuples, and sequences**. It aims to provide a strong understanding of how to use these data structures effectively and when to choose one over the other.

## Learning Objectives

By the end of this project, you are expected to be able to explain:

### General Concepts
- What are **lists** and how to use them
- Differences and similarities between **strings** and **lists**
- Common **list methods** and their usage
- How to use lists as **stacks** and **queues**
- How to create and use **list comprehensions**
- What are **tuples** and how to use them
- When to use tuples versus lists
- What is a **sequence**
- What is **tuple packing** and **sequence unpacking**
- How to use the `del` statement

## Requirements

- Python Scripts executed with **Python 3.8.5** on **Ubuntu 20.04 LTS**
- First line of all files: `#!/usr/bin/python3`
- All files should end with a new line
- Code must follow **pycodestyle 2.7.* standards**
- All files must be executable
- File length will be checked using `wc`
