
# Positive, Negative, or Zero

## Description
This program assigns a random signed number to the variable `number` each time it runs.  
The program then checks whether the number is positive, negative, or zero and prints the result.

## How It Works
- If the number is greater than 0 → the program prints that the number is **positive**
- If the number is equal to 0 → the program prints that the number is **zero**
- If the number is less than 0 → the program prints that the number is **negative**
