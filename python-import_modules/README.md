# Python - Modules

## Description

This project introduces the concept of **Python modules** and how to reuse code across different files.
It demonstrates how to import functions from other Python files, use command line arguments, and control script execution when modules are imported.

## Learning Objectives

By completing this project, I learned how to:

* Understand why Python programming is powerful and widely used
* Import functions from another Python file
* Use functions from imported modules
* Create and organize Python modules
* Use the built-in `dir()` function to explore module contents
* Prevent code from executing when a file is imported using `if __name__ == "__main__":`
* Use command line arguments in Python programs

## Resources

## Requirements

### General

* Allowed editors: **vi, vim, emacs**
* All files are interpreted/compiled on **Ubuntu 22.04 LTS** using **Python 3.10**
* All files end with a **new line**
* The first line of each file is:

```python
#!/usr/bin/python3
```

* A **README.md** file is included at the root of the project
* Code follows **pycodestyle (version 2.7)**
* All files are **executable**
* File length is tested using **wc**
